HOW TO USEdrop an agent in, run a scenario, benchmark it

1. Drop the agent into the victim container

python -m platform.eval.drop_agent --binary ./my_agent --start-cmd "/opt/clawdian_agent/my_agent --out /var/log/agent_alerts.jsonl" --container clawdian_victim

2. Run a scenario

python -m core.runner.executor scenarios/atomic/T1070_004_*.json

3. Score the benchmark

python -m platform.eval.benchmark reports/<run_id>_exec_log.json --agent-name "my-agent"

Agent constraints: victim container runs with network_mode: none and cap_drop: ALL. Use self-contained, file-writing agents. Commercial EDRs requiring network callbacks need the sidecar tier (see docs).
